001 : Yelp Camp Project

This is a full-stack web application project for a fictional start up called Yelp Camp. On this web application users can create their own campgrounds, add campground images, view, comment and rate on others' campgrounds.

Built with:

Front-end: EJS, Bootstrap, MapBax

Back-end: Node JS, Express, Mongo DB, Mongoose, Passport, Passport-local, Express-Session, Method-override, Connect-flash, Cloudinary
